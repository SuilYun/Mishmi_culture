# Mishmi Tribe Cultural Portal - Arunachal Pradesh

A cinematic, immersive digital showcase celebrating the history, lifestyle, festivals, language, and flora/fauna of the Mishmi Tribe of Arunachal Pradesh. Built with modern web styling, smooth animations, and interactive mapping.

## ✨ Features
- **Interactive Geographical Map**: Fully interactive Leaflet-based map spotlighting traditional districts (Dibang Valley, Lower Dibang Valley, Lohit, and Anjaw) with flying zoom transitions.
- **Lenis Smooth Scroll**: Silky-smooth scrolling experience.
- **Micro-Animations & Transitions**: Beautiful stagger effects, glassmorphic UI elements, and scroll progress tracking.
- **Flora & Fauna Gallery**: Showcases endemic species like the *Mishmi Takin* and the endangered *Mishmi Teeta* medicinal plant.
- **Visual Gastronomy Grid**: Highlights tribal delicacies like *Smoked Mithun Meat* and *Dun (Bamboo Rice)*.
- **Audio Language Guides**: Real-world pronunciations powered by the Web Speech API.

## 🛠️ Tech Stack
- **Framework**: React 18
- **Bundler**: Vite
- **Styling**: Tailwind CSS & Vanilla CSS
- **Interactions**: Framer Motion, Lucide React icons
- **Maps**: React Leaflet & Leaflet.js

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation & Local Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173` in your browser.

### Build for Production
To generate a compiled production build:
```bash
npm run build
```
The optimized bundle will be created inside the `dist` folder.
